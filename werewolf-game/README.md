# 狼人杀（没有法官时 · 浏览器联机）

这是一个**纯文字 + 实时同步**的狼人杀小程序：一个人当房主（上帝）控流程，其他人用手机/电脑浏览器加入同一房间即可玩。

本项目按你描述的玩法做成了 **线下随机发牌模式**：

- 房主配置本局人数与板子（角色与数量）
- 玩家进房后选择座位号
- 房主一键开始后，程序按板子**随机发牌并私发身份**
- 夜晚各身份在手机上完成操作，房主点击结算
- **先竞选警长，再公布昨夜信息**

## 运行方式

在目录 `werewolf-game/` 下执行：

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python app.py
```

然后在浏览器打开：

- 本机：`http://localhost:8080`
- 局域网：让同一 Wi‑Fi 的朋友打开 `http://你的电脑IP:8080`

## 使用说明（房主/玩家）

- 房主：
  - 创建房间 → 等所有人加入
  - 先配板子（神职/狼牌/民牌），**人数=角色数量合计**（系统自动计算）
  - 板子配好后，大家在页面里**选座位号**
  - 点“开始游戏（进入第一夜）”，会按座位随机发牌并私发身份
  - 点“开始游戏（进入第一夜）”
  - 夜晚：等大家提交完，点“结算夜晚（进入警长竞选）”
  - 警长：等大家投票，点“结算警长（并公布昨夜信息）”
  - 白天：等大家投票，点“结算投票（进入下一夜）”
  - 若出现猎人死亡，会进入“猎人开枪”阶段，由房主执行开枪
- 玩家：加入房间 → 选择座位 → 等待随机发牌 → 按阶段提交“刀人/验人/守护/救人/毒人/警长投票/白天投票”
